

## Install

User authentication with roles and permissions management system.

 - composer install
 - npm install
 - cp .env.example .env
 - php artisan key:generate
 - php artisan migrate
 - php artisan db:seed


 ## For admin access
  - admin : admin@admin.com
  - pass  : password
